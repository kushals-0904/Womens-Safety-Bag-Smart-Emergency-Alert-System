# Project Documentation

This folder contains the documentation details for the Women’s Safety Bag – Smart Emergency Alert System.

## Modules Description

### Arduino Uno
Acts as the central controller coordinating sensors and communication modules.

### GPS Module
Fetches real-time latitude and longitude data during emergency situations.

### GSM Module
Sends SMS alerts containing emergency messages and live location links.

### Sensors
- Panic Button for manual emergency triggering
- Vibration Sensor for detecting abnormal movements

## Testing Summary
- Code tested using Arduino IDE
- Alert logic validated using MATLAB
- Circuit behavior verified through simulation
